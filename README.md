# React MUI ECOM Template

## Description

This is a template for a React MUI ECOM project.

## Installation

```bash
npm install
```

## Features

- [x] React
- [x] Material UI

## Usage

```bash
npm start
```

## Build

```bash
npm run build
```

## Deployment

```bash
npm run deploy
```
